# Brain — AI-асистент для ресторатора

Лендинг для продукту **Brain** — AI-асистента, який стає правою рукою ресторатора: розумний, надійний, завжди поруч.

## 🌐 Жива версія

Сторінка публікується через GitHub Pages.
Після активації Pages у налаштуваннях репозиторію вона буде доступна за адресою:

```
https://<ваш-github-username>.github.io/mozok-landing/
```

## 📁 Структура

```
mozok-landing/
├── index.html      # Сам лендинг (один самодостатній файл)
└── README.md       # Цей опис
```

Усе — HTML, CSS і шрифти — лежить в одному файлі `index.html`.
Жодних залежностей, збірки чи бекенду не потрібно.

## 🚀 Як запустити локально

Просто відкрийте `index.html` у браузері — і все працює.

## 🛠 Як опублікувати на GitHub Pages

1. Зайдіть у репозиторій на GitHub.
2. Відкрийте **Settings → Pages**.
3. У розділі **Source** оберіть гілку `main` і теку `/ (root)`.
4. Натисніть **Save**.
5. За 1–2 хвилини GitHub видасть посилання на живу сторінку.

## 📝 Технічний стек

- Чистий HTML + CSS (без фреймворків)
- Шрифт Inter з Google Fonts
- Адаптивна верстка (мобільна / планшетна / десктопна)
- Українська мова (`lang="uk"`)
